Что такое Java?
Java – широко используемый язык программирования для написания интернет-приложений. Миллионы приложений Java используются и сегодня. Java – это многоплатформенный, объектно-ориентированный и сетецентрический язык, который сам по себе может использоваться как платформа. Это быстрый, безопасный и надежный язык программирования для всего: от мобильных приложений и корпоративного ПО до приложений для работы с большими данными и серверных технологий.
Где используется Java?
  • Создание игр
  • Создание desktop приложений
  • Создание android приложений
  • Создание веб-приложений
  • Обработка больших данных
